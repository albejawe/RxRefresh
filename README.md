# RxRefresh 💊

تطبيق ويب تقدمي (PWA) مصمم خصيصاً لمساعدة الصيادلة على استرجاع وتحديث معلوماتهم الدوائية بأسلوب تفاعلي ممتع يعتمد على نظام التكرار المتباعد (Spaced Repetition System).

## المميزات ✨
- **PWA كامل**: يمكن تثبيته على الهواتف والعمل دون اتصال بالإنترنت (Offline Support).
- **تصميم Glassmorphism**: واجهة جذابة داكنة بالكامل (Dark Mode) تريح العين مع حركات انسيابية (Framer Motion).
- **نظام التكرار المتباعد (SRS)**: تأجيل ذكي للبطاقات التي تُخطئ بها لمراجعتها في أوقات لاحقة.
- **إحصائيات متقدمة**: تتبع الـ Streak، والنقاط، ونسبة النجاح عبر رسوم بيانية (Recharts).
- **إشعارات محلية**: تنبيهات يومية بمواعيد المراجعة دون الحاجة لسيرفر خارجي.

## التقنيات 🛠️
- React 18 + Vite
- Tailwind CSS (Design Tokens)
- Framer Motion
- React Router
- LocalStorage (State Management)
- Vite PWA Plugin
